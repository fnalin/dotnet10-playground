# .NET 10 Playground

📚 Repositório de estudos e experimentações com **.NET 10**

## 📋 Descrição

Este repositório contém projetos, exemplos de código e experimentações relacionados ao aprendizado de **.NET 10**. É um espaço para explorar as novas funcionalidades, APIs e padrões que o framework oferece.

## 🎯 Objetivo

- 🔍 Explorar recursos e funcionalidades do .NET 10
- 💡 Testar novos padrões e práticas recomendadas
- 📖 Documentar aprendizados e descobertas
- 🚀 Implementar exemplos práticos de uso

## 📂 Estrutura do Repositório

```
dotnet10-playground/
├── README.md                 # Este arquivo
├── [Projetos]              # Projetos específicos de estudo
└── [Exemplos]              # Exemplos de código
```

## 🛠️ Tecnologias

- **.NET 10**
- C# (versão mais recente)
- Ferramentas de desenvolvimento padrão do .NET

## 🚀 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/[seu-usuario]/dotnet10-playground.git
   ```

2. Navegue até a pasta desejada:
   ```bash
   cd dotnet10-playground
   ```

3. Restore e execute:
   ```bash
   dotnet restore
   dotnet run
   ```
